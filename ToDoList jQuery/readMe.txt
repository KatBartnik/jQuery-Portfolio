Table of content:
I) basic functions
II) app structure
III) comments

I) 
Adding- you can add an entry by pushing 'enter' button.
Removing- you can remove an entry by clicking 'trash icon' when mousing over the entry.
Line through (to mark as done). You can change an entry status from standard to line-through (visual effect) to make it look like it's crossed out. You can change it by mouse clicking.
  
II) 
index.html- an executable file for the project
style.css- contains all CSS
main.js- jQuery methods
